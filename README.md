A comprehensive Hospital Management System database built using MySQL that manages patients, doctors, appointments, admissions, billing, pharmacy, laboratory services, staff, and ambulance operations. The project demonstrates relational database design, normalization, joins, triggers, views, stored procedures, indexing, and data integrity through SQL constraint    
Key Features
👨‍⚕️ Doctor and Department Management
🧑‍🤝‍🧑 Patient Registration and Medical Records
📅 Appointment Scheduling
🏥 Room Allocation and Admissions
💊 Pharmacy & Medicine Inventory Management
🧪 Laboratory Test Orders and Results
🚑 Ambulance Request Management
💳 Billing and Payment Tracking
👩‍⚕️ Staff and Shift Management
⭐ Patient Feedback System
🔐 User Authentication and Role Managemen
